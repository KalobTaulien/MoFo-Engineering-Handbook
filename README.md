# Engineering Discussions

## Dependencies:
- Ruby 2.1.0 (probably managed by [rbenv](https://github.com/sstephenson/rbenv)
or rvm)
- Node > 0.10.0
- [Bundler](http://bundler.io)

## Install:

```bash
npm install
```

## Run:

```bash
npm start
```

## Test: (there are no tests yet)

```bash
grunt check
```

## Deploy:
